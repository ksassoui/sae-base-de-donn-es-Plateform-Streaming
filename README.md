# SAE Base de Données - Plateforme de Streaming

## Contexte

Projet réalisé dans le cadre du module Bases de Données.

Le sujet consistait à modéliser une plateforme de streaming.

---

## Besoin

Concevoir une base de données permettant de gérer :

- les utilisateurs
- les abonnements
- les contenus
- les catégories
- l'historique de visionnage

---

## Technologies utilisées

- PostgreSQL
- SQL
- Modélisation de données

---

## Compétences mobilisées

- Gestion des données
- Modélisation
- Requêtes SQL
- Analyse des besoins

---

## Réalisations

- Modèle conceptuel
- Modèle logique
- Création des tables
- Insertion des données
- Requêtes d'analyse

---

## Difficultés rencontrées

- Gestion des relations complexes
- Normalisation des données
- Conception du schéma relationnel

---

## Résultat

Une base cohérente permettant de représenter le fonctionnement d'une plateforme de streaming.

---

## Si c'était à refaire

J'ajouterais un système de recommandations basé sur l'historique des utilisateurs.
